## Sobre Projecta

**Projecta** es mi **Trabajo Final de Grado (TFG)** para la especialización en **Desarrollo de Aplicaciones Web**. Se trata de una **plataforma web completa de gestión de proyectos** que permite a los equipos colaborar, asignar tareas y gestionar el progreso de sus iniciativas de forma eficiente.

## Características principales

- **Gestión de usuarios y equipos** con roles y permisos diferenciados
- **Creación y organización de proyectos** con seguimiento detallado
- **Sistema de tareas** con asignación, estados y prioridades
- **Comentarios y comunicación** en tiempo real dentro de proyectos
- **Registro de actividades** y auditoría de cambios
- **Autenticación segura** y gestión de sesiones en base de datos

## Stack tecnológico

Este proyecto ha sido desarrollado con un **stack moderno y profesional**:

- **Backend:** Laravel 12 + PHP 8.2+ con Blade templating
- **Frontend:** HTML5, CSS3, JavaScript + SASS para estilos avanzados
- **Base de datos:** SQLite (por defecto) o MySQL
- **Bundler:** Vite para optimización de assets
- **Testing:** Suite completa de tests automatizados
- **Control de versiones:** Git
